Projects

Enhanced Multi-Model Feature Fusion for categorizing Diabetic Retinopathy Severity Level

Description : The project aims to improve the accuracy of diagnosing diabetic retinopathy (DR) severity levels by using a multi-model fusion approach. It combines the strengths of two pre-trained models, VGG19 and ResNet50, leveraging a cross-pooling technique to extract and integrate crucial image features from retinal fundus images. The proposed model, tested on the Kaggle APTOS 2019 dataset, achieved an accuracy of 83.33%, demonstrating significant potential for early DR intervention and effective classification of different DR severity stages.
